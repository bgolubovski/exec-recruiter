    Status definitions
    Pending — outreach sent, awaiting response. Day-5 / Day-7 / Day-10 bumps fire automatically based on send date.
    Prepped — packet built (CV / cover letter / Connect-Note ready) but not yet sent. Awaiting user greenlight. No send date.
    Replied — substantive response received (interview invite, recruiter call, hiring-manager engagement). Active conversation.
    Rejected — employer formally declined the application (template rejection email, ATS auto-reject, recruiter "no" reply, etc.). Engagement happened, outcome was no.
    Skipped — never pursued substantively. Pre-emptive decision based on hard blocker (clearance, on-site geo, hard domain gate), low fit score, role withdrawn by employer before applying, or downgraded after pre-send check.
    Closed — reserved for late-stage process ends (interviewed, took home assignment, finalist round) where the engagement got past initial screen but ended without offer. Distinct from rejected (which is upfront-no) and replied (which is still active).

    How to use this dashboard

    Cadence: Open this every other day. The "Today's Actions" panel surfaces roles where Day 5-7 follow-up bumps are due (yellow), Day 10 final actions are due (orange), and overdue roles (red).

    Sorting: Click any column header. Default sort is by Score descending.

    Filters: Use the pill filters to narrow by status or channel. Search box matches across role/company/contact.

    File links per row:

    • JD → opens local jd.md (saved at fetch time)
    • JD↗ → opens LinkedIn listing (live, may be closed)
    • CV → opens tailored PDF in your folder
    • DM → opens the SEND-{name}.txt that was paste-sent
    • Status → opens status.md with the full role timeline + follow-up cadence

    To update: When a role replies, edit the JS data block in this file (look for const ROLES = [...] near the bottom) and change the status field. The dashboard recomputes stats and "Today's Actions" automatically.

    To add a new role: Append to const ROLES = [...] with the same shape as existing rows. Save and reload.
